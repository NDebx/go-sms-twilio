# Go send SMS via REST and Twilio 

You can send SMS to your end user/client via Twilio, using the Go language. The REST endpoint are powered by the Gin framework.

## Author

- [@NileshDebix](https://github.com/NileshDebix)


## Installation/Usage

Step 1: First install Golang on your computer and clone the repo

Step 2: Edit the config.yml file and insert your Twilio account details

Step 3: Run the API
```bash
go run mailsms-api.go
```
